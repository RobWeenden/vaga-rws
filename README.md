# projeto doacao de sangue

###### Ferramentas de desenvolvimento novo comentario
* Java v21</br>
* Spring boot v3.3.6</br>
* Docker </br>
* Docker compose v3.8 </br>
* Postgres latest

###### Como rodar o projeto
1. Instale o docker na maquina link: [docker](https://docs.docker.com/engine/install/)
2. Baixe o projeto
3. Va na pasta raiz onde possui o arquivo **docker-compose.yml**

###### Rodar o projeto no docker
1. No diretorio raiz do projeto execute: **docker-compose up**

### Para acessar o projeto
1. Url: [localhost:8081](http://localhost:8081/)
2. Para realizar o usuário pode criar uma novo usuario ou utilizar o usuário já pré-cadastrado para teste.
